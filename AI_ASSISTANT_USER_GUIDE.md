## 🚀 Usage

### Sending Message
1. Type your question
2. Press **Enter** or click send button ⭕
3. Wait for "AI is thinking..." animation
4. Get response
5. Message saves automatically

### Viewing History
1. Close AI Assistant
2. Reopen it
3. **All previous messages appear!** ✅
4. Scroll to see older messages

### Copying Responses
1. **Hover** over any AI message
2. Click copy icon 📋
3. See "✓ Copied"
4. Paste with Ctrl+V

### Giving Feedback
1. **Hover** over any AI message
2. Click 👍 (helpful) or 👎 (not helpful)
3. Feedback recorded
4. Helps improve future responses

### Starting Fresh
1. Click **↺** button at top
2. Confirms: "New Chat"
3. Clears history for this session
4. Shows greeting message again

## 💾 Data Storage

**All your chat is saved in browser storage:**
- Per user (each user has own history)
- Automatically saved on every message
- Loads automatically when you return
- Shows in browser DevTools → Application → Local Storage

## ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| **Enter** | Send message |
| **Shift+Enter** | New line in message |
| **Escape** | Close assistant (usually) |
| **Ctrl+V** | Paste (after copying) |

## 🔧 Troubleshooting

### Messages Not Showing
- Check browser console (F12)
- Look for error messages
- Try refreshing page
- Clear browser cache if needed

### Scrollbar Not Visible
- Try scrolling in chat area
- Check if chat is full (needs scroll)
- Use mouse wheel or trackpad
- On mobile, use touch swipe

### Copy Not Working
- Check browser permissions
- Try again with another message
- Restart browser if needed

### Chat Not Saving
- Check localStorage is enabled
- Look at browser console for errors
- Verify you're logged in
- Try "New Chat" and ask again

## 📱 Mobile/Tablet

- Chat works on all screen sizes
- Scrollbar works on touch devices
- Use touch scroll to see history
- Tap copy button to copy
- Tap send button or Enter key

## 🎓 Advanced Tips

### Organizing Conversations
- Use "New Chat" to separate topics
- Each new chat starts fresh
- Previous chats remain saved
- Login as different user = different history

### Better Questions
- Be specific: "What documents needed for residential plot?"
- Ask follow-ups: "Can you explain that more?"
- Request formats: "List 3 key documents"
- Ask comparisons: "How does survey differ from mapping?"

### Using Feedback
- Mark helpful responses 👍
- Mark unclear responses 👎
- Helps AI learn your preferences
- Future responses improve
